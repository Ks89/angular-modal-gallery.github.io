Output Events example

Minimal and simple modal gallery example with output events logged in console.


Live Demo




Live demo results
Output $event value (Event payload)
(hasData)
{{imageLoaded | json}}
(close)
{{closeImageModal | json}}
(show)
{{visibleIndex | json}}
(firstImage)
{{isFirstImage | json}}
(lastImage)
{{isLastImage | json}}


Code


      <modal-gallery [modalImages]="imagesArray"
                     (hasData)="onImageLoaded($event)"
                     (close)="onCloseImageModal($event)"
                     (show)="onVisibleIndex($event)"
                     (firstImage)="onIsFirstImage($event)"
                     (lastImage)="onIsLastImage($event)">
      </modal-gallery>

Details


Input Value Description
[modalImages]
Array<Image>
 Array of Image objects
[imagePointer]
undefined
 It will display the entire gallery
[downloadable]
undefined
(false by default)
 Not downloadable at all
[description]
undefined
 It will use the default description
[showDownloadButton]
undefined
(false by default)
 Download button is hidden
[showExtUrlButton]
undefined
(false by default)
 External url button is hidden
