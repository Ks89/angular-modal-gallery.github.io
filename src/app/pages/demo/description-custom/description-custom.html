Description custom example

Minimal and simple modal gallery with a custom description.

So imageText, numberSeparator and beforeTextDescription will be replaced with the provided values, but the final description will be the value defined inside the Image object


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [description]="customDescription"></modal-gallery>

Details


Input (.html) Value (.ts) Notes
[modalImages]="var"
Array<Image>
 Array of Image objects
[imagePointer]="var"
undefined
 It will display the entire gallery
[downloadable]="var"
undefined
(false by default)
 Not downloadable at all
[description]="var"
imageText: 'Look this image ',
numberSeparator: ' of ',
beforeTextDescription: ' => '
It will customize the description using the input parameters: imageText, numberSeparator and beforeTextDescription
[showDownloadButton]="var"
undefined
(false by default)
 Download button is hidden
[showExtUrlButton]="var"
undefined
(false by default)
 External url button is hidden

Output (.html) Return (.ts)
(close)="funct($event)" funct(e: ImageModalEvent)
(show)="funct($event)" funct(e: ImageModalEvent)
(firstImage)="funct($event)" funct(e: ImageModalEvent)
(lastImage)="funct($event)" funct(e: ImageModalEvent)
(hasData)="funct($event)" funct(e: ImageModalEvent)
