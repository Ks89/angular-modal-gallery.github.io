Description custom example

Minimal and simple modal gallery with a custom description.

So imageText, numberSeparator and beforeTextDescription will be replaced with the provided values, but the final description will be the value defined inside the Image object


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [description]="customDescription"></modal-gallery>

Details


Input (.html) Value (.ts)
modalImages
Value: Image[]
Array of Image objects
imagePointer
Value: undefined
It will display the entire gallery
downloadable
Value: undefined
(false by default)
Not downloadable at all
description
Value: object with these properties
imageText: 'Look this image ', numberSeparator: ' of ', beforeTextDescription: ' => '
It will customize the description using the input parameters
showDownloadButton
Value: undefined
(false by default)
Download button is hidden
showExtUrlButton
Value: undefined
(false by default)
External url button is hidden
