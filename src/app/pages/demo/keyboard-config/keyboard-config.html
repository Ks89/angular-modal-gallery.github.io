keyboardConfig
Value: {esc: 81, left: 40, right: 38}

This configuration will map actions to other keyboard's keys. So, if you want to change the key to close modal gallery, you can use this property. In this example:

  • close gallery: q (instead of the default esc)
  • previous image: ↓ (instead of the default ←)
  • next image: ↑ (instead of the default →)
