Observable Simple example

Minimal and simple modal gallery example.


Live Demo (Delay of 300ms)




Code


      <modal-gallery [modalImages]="imagesObservable"></modal-gallery>

Details


Input (.html) Value (.ts)
modalImages
Value: Observable< Image[] >
Observable of an Array of Image objects
imagePointer
Value: undefined
It will display the entire gallery
downloadable
Value: undefined
(false by default)
Not downloadable at all
description
Value: undefined
It will use the default description
showDownloadButton
Value: undefined
(false by default)
Download button is hidden
showExtUrlButton
Value: undefined
(false by default)
External url button is hidden
