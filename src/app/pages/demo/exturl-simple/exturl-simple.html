ExternalURL Simple example

Minimal and simple modal gallery example with external url button.


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [showExtUrlButton]="true">
      </modal-gallery>

Details


Input (.html) Value (.ts) Notes
[modalImages]="var"
Array<Image>
 Array of Image objects
[imagePointer]="var"
undefined
 It will display the entire gallery
[downloadable]="var"
undefined
(false by default)
 Not downloadable at all
[description]="var"
undefined
 It will use the default description
[showDownloadButton]="var"
undefined
(false by default)
 Download button is hidden
[showExtUrlButton]="var"
true
 External url button is visible

Output (.html) Return (.ts)
(close)="funct($event)" funct(e: ImageModalEvent)
(show)="funct($event)" funct(e: ImageModalEvent)
(firstImage)="funct($event)" funct(e: ImageModalEvent)
(lastImage)="funct($event)" funct(e: ImageModalEvent)
(hasData)="funct($event)" funct(e: ImageModalEvent)
