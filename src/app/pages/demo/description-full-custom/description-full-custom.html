Description full custom example

Minimal and simple modal gallery with a full custom text description.

The entire description will be replaced with the provided text.


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [description]="customFullDescription"></modal-gallery>

Details


Input (.html) Value (.ts) Notes
[modalImages]
Image[]
 Array of Image objects
[imagePointer]
undefined
 It will display the entire gallery
[downloadable]
undefined
(false by default)
 Not downloadable at all
[description]
customFullDescription: 'Custom description of the current visible image'
It will use the provided value
[showDownloadButton]
undefined
(false by default)
 Download button is hidden
[showExtUrlButton]
undefined
(false by default)
 External url button is hidden

Output (.html) Return (.ts)
(close) ImageModalEvent)
(show) ImageModalEvent)
(firstImage) ImageModalEvent)
(lastImage) ImageModalEvent)
(hasData) ImageModalEvent)
