Download Advanced example

Minimal modal gallery example with downloadable images ONLY with ctrl+s/cmd+s.


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [downloadable]="true"
                     [showDownloadButton]="false">
      </modal-gallery>

Details


Input (.html) Value (.ts)
modalImages
Value: Image[]
Array of Image objects
imagePointer
Value: undefined
It will display the entire gallery
downloadable
Value: true
Downloadable with ctrl+s and cmd+s
description
Value: undefined
It will use the default description
showDownloadButton
Value: false
Download button is hidden
showExtUrlButton
Value: undefined
(false by default)
External url button is hidden
