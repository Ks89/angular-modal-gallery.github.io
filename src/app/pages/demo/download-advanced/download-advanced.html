Download Advanced example

Minimal modal gallery example with downloadable images ONLY with ctrl+s/cmd+s.


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [downloadable]="true"
                     [showDownloadButton]="false">
      </modal-gallery>

Details


Input (.html) Value (.ts) Notes
[modalImages]
Image[]
 Array of Image objects
[imagePointer]
undefined
 It will display the entire gallery
[downloadable]
true
 Downloadable with ctrl+s and cmd+s
[description]
undefined
 It will use the default description
[showDownloadButton]
true
 Download button is hidden
[showExtUrlButton]
undefined
(false by default)
 External url button is hidden

Output (.html) Return (.ts)
(close) ImageModalEvent)
(show) ImageModalEvent)
(firstImage) ImageModalEvent)
(lastImage) ImageModalEvent)
(hasData) ImageModalEvent)
