Download Simple example

Minimal and simple modal gallery example with downloadable images.


Live Demo




Code


      <modal-gallery [modalImages]="imagesArray"
                     [downloadable]="true"
                     [showDownloadButton]="true">
      </modal-gallery>

Details


Input Value Description
[modalImages]
Array<Image>
 Array of Image objects
[imagePointer]
undefined
 It will display the entire gallery
[downloadable]
true
 Downloadable with ctrl+s and cmd+s
[description]
undefined
 It will use the default description
[showDownloadButton]
true
 Download button is visible
[showExtUrlButton]
undefined
(false by default)
 External url button is hidden

Output Return Description
(close) ImageModalEvent Ignored in this simple example
(show) ImageModalEvent Ignored in this simple example
(firstImage) ImageModalEvent Ignored in this simple example
(lastImage) ImageModalEvent Ignored in this simple example
(hasData) ImageModalEvent Ignored in this simple example
